eTQF3 Form

Instructions

The coordinators of all courses are required to submit a TQF3 before the start of the semester. The TQF3 provides information about the course's aims, content structure, teaching and assessment methods, skills development, guidelines, and other information that students may need to achieve the course objectives/learning outcomes at the requisite level. It is important, therefore, to share this document with students at the start of the course. The eTQF system allows you to easily create, modify, reuse, submit and share the TQF3 for your course. Formatted TQF3 are created from a Microsoft Word template (.docx) that is the same for the whole School of Liberal Arts. But the most imporant file that you need to keep and submit is the eTQF3, which is a json data file (.json). The system will automatically validate your eTQF3 when you generate it. You can save an invalid draft to work on it at a later time. You can also reuse eTQF3 from other courses or semesters in order to speed up the process of filling up the information for your course.

To create a new TQF3:

  1. Fill up all the required fields (*) and the optional fields that are relevant for your course. Hover over question icons to obtain guidelines on how to fill up the different fields.
  2. Click on Generate TQF3 when you are done.
  3. The form will be automatically validated.
  4. If it is valid, two files will be downloaded: formatted TQF3 (.docx) and eTQF3 (.json). If it is not valid, only the json file will be downloaded. Keep this draft and upload it at a later time to continue editing.

eTQF5 Form

Instructions

The coordinators of all courses are required to submit a TQF5 at the end of the semester. The TQF5 reports the teaching and learning outcomes of a course, including student academic results, number of students retained, course management problems, course evaluation by students, and recommendations for further improvement and development. After the semester has ended, the Curriculum Committee will evaluate and review the TQF5 submitted for all the courses. The eTQF system allows you to easily create, modify, reuse, and submit the TQF5 for your course. Formatted TQF5 are created from a Microsoft Word template (.docx) that is the same for the whole School of Liberal Arts. But the most imporant file that you need to keep and submit is the eTQF5, which is a json data file (.json). To start the process of creating an eTQF5, you need to provide a valid eTQF3 for the course you are reporting. The system will automatically validate your eTQF5 when you generate it. You can save an invalid draft to work on it at a later time.

To create a new TQF5:

  1. Upload a valid eTQF3 file (.json) for the course you want to report the TQF5.
  2. Fill up all the required fields (*) and the optional fields that are relevant for your course. Hover over question icons to obtain guidelines on how to fill up the different fields.
  3. Click on Generate TQF5 when you are done.
  4. The form will be automatically validated.
  5. If it is valid, two files will be downloaded: formatted TQF5 (.docx) and eTQF5 (.json). If it is not valid, only the json file will be downloaded. Keep this draft and upload it at a later time to continue editing.

eTQF Signature

Instructions

Once validated, eTQFs can be signed by any registered member of staff. The same eTQF cannot be signed by the same person more than once. But an unlimited number of signatures of different staff members, such as coordinators, instructors, secretaries, or supervisors, can be added to the same eTQF. To be able to submit the eTQF of a course, it must have been signed by at least one of the coordinators of that course. You will not be able to sign an eTQF that has not been validated by using the Generate TQF3/TQF5 buttons in the previous tabs.

To sign an eTQF:

  1. Upload a valid eTQF3 or eTQF5 file (.json).
  2. Select your name from the dropdown menu.
  3. Introduce your secret password.
  4. Press to sign the eTQF.
  5. Two files will be downloaded: signed eTQF (.json) and signed formatted TQF (.docx).

eTQF Verification

Instructions

eTQF signatures use a private/public key cryptographic system that allows anyone to verify the validity of a signature by uploading the signed eTQF here. Verification ensures that the eTQF has been signed by the staff member whose id and email appear in the signature.

To verify an eTQF:

  1. Upload a signed eTQF3 or eTQF5 file (.json).
  2. Select the signature to verify from the dropdown menu (only the names of the staff members who have signed the eTQF appear on the menu).
  3. Press the button 'Verify eTQF' to verify the signature.

eTQF Submission

Instructions

Coordinators need to submit the TQF3 and TQF5 for their courses before the deadline set by the School of Liberal Arts. Once the eTQF3 or eTQF5 has been signed by at least one of the coordinators, it can be submitted to the database that the Liberal Arts administrative staff will use to check the submission. Only the coordinators of a course can submit the eTQF of that course to the database. Data on all submissions, including links to the submitted documents, are accessible for all staff in the Data tab of the eTQF system. Once submitted, TQF3 are also available to be downloaded by students and anyone else by using a public URL: "https://etqf.herokuapp.com/download?[program_code][course_code][year][semester]". For example, the link to download the TQF3 for a course of the Bachelor of English (program code 633100601) with the code 1006110 for the year 2022 and first semester would be: "https://etqf.herokuapp.com/download?633100601100611020221". This URL only works for eTQF3s that have been successfully submitted using this system and allows anyone to download the formatted Word version of the TQF3.

To submit your eTQF:

  1. Upload a signed eTQF3 or eTQF5 file (.json).
  2. Select your name in the dropdown menu.
  3. Press the button 'Submit eTQF'.

eTQF Registration

Instructions

Before you can sign, submit and view submitted eTQFs, you need to have a public key stored in the staff directory. You can generate this key by introducing a secret password. This password will serve to generate a public key using a secure cryptographic algorithm. Your password WILL NOT BE STORED anywhere and cannot be retrieved in any way from the public key. For your security, make sure to remember your password and do not share it with anyone. If you forget or want to change your password, you can always generate a new public key using this same form. After generating it, you will receive an email with a one-time password token (OTP) to complete the registration.

To register in the eTQF system:

  1. Select your name in the dropdown menu.
  2. Enter a secret password at least 6 characters long.
  3. Press the button 'Generate public key'.
  4. Enter the token received in your email.
  5. Press the button 'Confirm registration'.

eTQF Data

Instructions

If you are a registered staff member, you can view all the submitted eTQF3 and eTQF5 for a particular program, year and semester filling up the form below. Please remember that the TQF3 of all submitted courses is also available for direct download by students using the following link: "https://etqf.herokuapp.com/download?[program_code][course_code][year][semester]". For example, the link to download the TQF3 for a course of the Bachelor of English (program code 633100601) with the code 1006110 for the year 2022 and first semester would be: "https://etqf.herokuapp.com/download?633100601100611020221". This link can be publicly shared and can be used to download the formatted (.docx) TQF3 of the course.

To view the submissions:

  1. Select a program, year and semester.
  2. Select your name from the staff dropdown menu.
  3. Introduce your password.
  4. Press the button 'View data'.
