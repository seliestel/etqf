Valid Action Verbs

The following verbs, derived from Bloom's Taxonomy, may be used when writing the STUDENT LEARNING OUTCOMES on section 3 of the eTQF3. Start the learning outcomes by selecting action verbs adequate to the cognitive level required. Verbs not included in this list will not be validated. Please note that the verbs in the heading of each level of Bloom's taxonomy (e.g. 'understand') are NOT valid verbs.
